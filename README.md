# Translation System UI

نظام إدارة ترجمة متكامل مبني باستخدام Next.js و Tailwind CSS.

## متطلبات التشغيل

1. **Node.js**: تأكد من تثبيت Node.js (الإصدار 18 أو أحدث) على جهازك.
2. **Backend API**: يجب أن يكون الـ API يعمل على الرابط `http://localhost:5296`.

## خطوات التشغيل

1. **تثبيت المكتبات**:
   افتح التيرمينال في مجلد المشروع وقم بتنفيذ الأمر التالي:
   \`\`\`bash
   npm install
    & "C:\Users\Ahmed\Downloads\node-v24.11.1-win-x64\node-v24.11.1-win-x64\npm.cmd" run dev
   \`\`\`

2. **إعداد المتغيرات البيئية (اختياري)**:
   المشروع مهيأ افتراضياً للاتصال بـ `http://localhost:5296/api`.
   إذا أردت تغيير الرابط، قم بإنشاء ملف `.env.local` وأضف السطر التالي:
   \`\`\`env
   NEXT_PUBLIC_API_URL=http://your-api-url/api
   \`\`\`

3. **تشغيل المشروع**:
   لتشغيل واجهة المستخدم في وضع التطوير:
   \`\`\`bash
   npm run dev
   \`\`\`

4. **الوصول للتطبيق**:
   افتح المتصفح على الرابط: `http://localhost:3000`

## هيكلية المشروع

- `/app`: صفحات التطبيق (Next.js App Router)
  - `/manager`: لوحة تحكم المدير
  - `/supervisor`: لوحة تحكم المشرف
  - `/reviewer`: لوحة تحكم المدقق
  - `/translator`: لوحة تحكم المترجم
  - `/data-entry`: لوحة تحكم مدخل البيانات
- `/components`: مكونات واجهة المستخدم
- `/lib`: دوال مساعدة و API Client
- `/types`: تعريفات TypeScript

## تسجيل الدخول

يمكنك استخدام أي حساب موجود في قاعدة بيانات الـ Backend الخاص بك، وسيقوم النظام تلقائياً بتوجيهك للوحة التحكم المناسبة بناءً على دور المستخدم (Role).
